Este proyecto es un conversor de monedas en Java, desarrollado como desafío práctico.
Utiliza la API ExchangeRate-API junto con la biblioteca Gson para consumir datos en formato JSON y obtener tasas de cambio en tiempo real.

Incluye:
	•	Consumo de API con HttpClient
	•	Análisis de JSON usando Gson
	•	Conversión entre monedas seleccionadas
	•	Interfaz por consola con menú interactivo
	•	Clases: Principal, Conversor, ClienteAPI
